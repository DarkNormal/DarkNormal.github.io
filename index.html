Skills and Familiar Technologies

Android Development

One of my favourite platforms for development is mobile, specifically Android. My strong core skills in OOP languages such as Java and C# allow me to rapidly develop applications in a full-stack process. Google's extensive collection of APIs provide me with a set of tools to integrate my applications seamlessly into the Android universe and my server logic is developed with Node.js or C#.

I also enjoy UI design, which I'm currently taking a course in to improve my skills. I belive having the ability to understand the possibilites and limitations of both front-end and back-end mobile development is essential

C#, Java and SQL

Some of my strongest skills when it comes to programming is with Object-Orientated languages. The majority of my experience is with Java and C#, both being very similar in terms of syntax and practice. Many of the projects that I've been involved with include some form of database or data persistence, which is where my strong knowledge of SQL and databases shines. Hibernate and Entity Framework are of high importance for me as it increases the functionality of a program tenfold.

Web & Graphic Design

In the last number of years, I've created a number of websites for various clients using different technologies. Some of these include the IT Tallaght Snowsports Society, MyDealDoc.ie, and The Canal Cafe Vicarstown. I have used WordPress, HTML, CSS and JavaScript for these websites, and I have hosted and managed various other websites. The experience I gained creating and managing these sites via Microsoft Azure, AWS and Blacknight gave me invaluable knowledge to the process.

I have also taken up graphic design and illustration to gain experience with these techniques, as web design regularly requires some form of graphic design incorporated into it. I completed 2 courses in Adobe Illustrator and Photoshop last year, which gave me a strong understanding of the basic uses of both applications.

GitHub Pages

By committing all the website files to a GitHub repository, a website can be accessed using GitHub Pages. The repository must be called your_github_username.github.io, and this will allow the website to be created. This process is just as easy as using Azure, and somewhat better as it enables automatic source-control.
