Site Breakdown

I'm a Computing student, and I did not use any template for this website. I did use an alternative to Bootstrap - Material Design Lite by Google, for the navigation bar / drawer and for some other styles, but I implemented these myself. The page-preloader is taken from a site which I thought would be a nice element if pre-loading the page was required. The image background on the home page can be slow to load on some connections, but the loader itself is not required. Some Javascript / JQuery was taken from various CodePens and has been marked in the source files.

GitHub Pages

By committing all the website files to a GitHub repository, a website can be accessed using GitHub Pages. The repository must be called your_github_username.github.io, and this will allow the website to be created. This process is just as easy as using Azure, and somewhat better as it enables automatic source-control.
